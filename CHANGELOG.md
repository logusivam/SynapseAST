## [1.0.0] - 2026-07-10
### Added
- Professional Feature-Sliced Design (FSD) architecture client setup.
- Monaco Editor code workspace side-by-side with interactive AST React Flow visualizer.
- Debounced client-side parser compilation using `@babel/parser`.
- Bidirectional cursor sync between AST node graph and Monaco code ranges.
- Collapsible node subtrees and node filtering configurations.
- Animated SVG hamburger menus, custom Legend overlay panels, and dynamic pane resizers.
- Fully responsive layout configurations across Landing, Editor, Examples, and Glossary pages.
- Dynamic starting copyright calculations inside widgets/footer.
- FAQ accordion questions detailing ESTree standards and security considerations on LandingPage.
- Open-source repo assets: CODEOWNERS, issue guidelines ymls, Vitest + Playwright setups, and pre-push hooks.
- Static robots.txt, sitemap.xml, Vercel routing configs, and structured JSON-LD SEO optimizations.
